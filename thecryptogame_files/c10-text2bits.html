How can Text be converted to Binary?
The attached article in the previous post might have given you some idea into how can we do this.
How do we convert characters into binary? By using ASCII codes!
ASCII stands for American Standard Code for Information Interchange, and is a character-encoding scheme which uses 8 bits to represent 1 character. For ex: 01000001 represents the character 'A', 01000010 represents 'B', and so on for 2^8 = 256 characters.
The ASCII table for each of the 256 characters are given below. Note that many ascii characters are 'control'-characters like delete, linefeed, etc. and are 'non-printable'.

Note

While using the 'Tool'(in the other tab), you may find many 'valid' binary strings which don't have any 'readable'/'printable' text-equivalent!
Example: 00000000 is binary of 'null-character'; 00001000 is binary of 'backspace', 00100000 means 'space'. The binary equivalent of 01111111 is 'del' (or delete).
In short, when the binary input corresponds to a white-space-like character, the "Output String" area will not 'show' any text.
